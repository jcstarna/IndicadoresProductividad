REQUISITOS

Para poder ejecutar la aplicacion de control de tableros es necesario
instalar el driver del conversor y el Java Runtime Enviroment (JRE) version 8 (minimo).
Para ambos es necesario tener instalado Windows 7 o superior.



INSTALACION DE DRIVERS Y JRE

El archivo install.bat lanza los instaladores del JRE y los Drivers necesarios para ejecutar la aplicacion.

La instalacion se puede hacer manualmente en caso de que los instaladores no se lancen automaticamente.
Para instalar manualmente primero debe identificar si su sistema es de 32 bits o 64 bits.

Para sistemas de 32 bits ejecute:
    - jre-8u144-windows-i586.exe Para instalar el JRE
	- CP210xVCPInstaller_x86.exe en la carpeta CP210x_Windows_Drivers para instalar los Drivers del conversor
	
Para sistemas de 64 bits ejecute:
    - jre-8u144-windows-x64.exe Para instalar el JRE
	- CP210xVCPInstaller_x64.exe en la carpeta CP210x_Windows_Drivers para instalar los Drivers del conversor

	

EJECUCION APLICACION
Una vez instalado los elementos necesarios copie la carpeta "control-tableros" donde desea y ejecute el programa "start.exe"
para iniciar la aplicacion, el inicio puede tardar un par de segundos.
